# LinkedIn Enrichment Pipeline

## Features
- LinkedIn profile discovery using Serper
- Profile enrichment using PDL
- Role summarization using Gemini
- Excel input/output workflow

## Stack
- Python
- Serper API
- People Data Labs
- Gemini API
- Pandas

## Setup

1. Clone repo
2. Install dependencies
3. Add `.env`
4. Run `python main.py`
